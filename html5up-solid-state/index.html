QuaDraw

One of my dearest projects to date, Quadraw utilises the Quadtree data structure to provide an optimised drawing solution for its users by improving efficiency and time complexity. Users are also given the option to choose from multiple colors and the key feature is the 'toggle Quadtree' button which also serves educational purpose; it allows visual demonstration of how Quadtree works, hence making it easy for the users to understand. Most importantly, the Quadtree significantly improves the eraser function, showcasing its core application.

Epilepsy Care

The idea for Epilepsy Care emerged when our group of four students noticed a lack of resources for epilepsy patients in Pakistan, especially in Urdu, a language widely understood here. We developed this bilingual website to fill that gap, providing essential information and resources in both English and Urdu. The site features an emergency response plan and quizzes to test your epilepsy knowledge. It also connects users with local services, aiming to improve access to vital support and information regarding Epilepsy.

Maze Runner

As a computer science student, I embraced the classic beginner project: a maze generation game. Built using Pygame, this project employs Depth-First Search (DFS) to create a new maze each time and ensures a clear path from the player to the destination. Because, let’s face it, what’s more fitting for a first project than a game where you navigate through endless mazes?

RPS

Following a summer of learning HTML, CSS, and JavaScript, I applied my new skills by creating a Rock, Paper, Scissors game. As a fun feature, the selected option enlarges on hover, adding an interactive touch. Additionally, users can visualize the computer’s choice, enhancing the gameplay experience. This project allowed me to put my knowledge into practice and significantly boosted my web development skills. And on a side note, it felt really cool watching the code come to life!
